## DMIT 是什么：先建立一个基本认知

**DMIT**（dmit.io）是 2018 年在美国纽约注册的高性能 VPS 服务商，华人背景，中文客服，支付宝/微信/PayPal 都支持付款。目前运营三个数据中心：美国洛杉矶（LAX）、中国香港（HKG）、日本东京（TYO）。

区别于普通廉价 VPS 的地方，主要在两点：

一是线路。DMIT 自有带宽，和中国三大运营商（电信、联通、移动）直接对接，手里有 CN2 GIA、AS9929、CMIN2 这几条对国内用户来说含金量最高的线路。晚高峰不降速，丢包率控制得很稳。

二是硬件。全系标配 AMD EPYC 处理器，KVM 虚拟化，企业级 SSD——洛杉矶部分套餐已经升到最新的 EPYC 9005 平台（AN5）。

流量超出后不会直接断机，而是降速继续跑。T1 系列 WEE 套餐限速 50Mbps（洛杉矶限速 100Mbps），相当于"软性不限量"，这个设计对流量消耗大的用户来说挺友好的。

IP 被墙可以每 15 天免费换一次，这点在这个价位段里算是有诚意的政策。

**优惠码有使用限制吗？**

大部分优惠码要求季付或年付才能激活，月付不适用。每个账号每次只能使用一个优惠码，且大多数码是"循环折扣"，意味着续费也按折扣价，不存在续费涨价。

**DMIT 适合建网站吗？**
